한페이지당 글 10개
페이지들의 그룹 : 5페이지를 하나의 그룹
총 156개의 글156/10 => 15
나머지 6개 +. 1페이지 총 16페이지로 나뉨

한개의 그룹당 5페이지
16/5=3 => 3개의 그룹
나머지 1 + +1개의 그룹 총 4개의 그룹



첫번째 그룹(1) => 1 2 3 4 5
두번째 그룹(2) => 6 7 8 9 10
세번째그룹(3) => 11 12 13 14 15
네번째그룹(4) => 16

내가보고있는 페이지(2번째 그룹의 7페이지를 보고있다면)
< 를 클릭하면 현재페이지 -1 = 6
<< 를 클릭하면 현재그룹-1 => 2-1 => 이전그룹의 마지막 페이지

> 를 클릭하면 현재페이지 +1 => 8
>>를 클릭하면 현재그룹 +1 => 3그룹 => 이 그룹의 첫 페이지
