pre요소

일반적인 요소들은 탭 스페이스 엔터가 아무리 요소내에 많아도 하나로 처리한다
프로그램 등의 코드를 화면에 나타내야할 경우에는 pre 태그가 효과적이다

pre가 없는 코드 => 보통 이런식으로 작성하지 않는다

Public class Test {
    public static void main(String[] args){
        Sysytem.out.pritntln("Hello World");
     }
}

pre가 있는 코드

        Public class Test {
        public static void main(String[] args){
        Sysytem.out.pritntln("Hello World");
             }
        }
