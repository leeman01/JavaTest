***** jquery 소개

- 2006년 모질라의 개발자인 존 레식이 개발한 자바스크립트 라이브러리
- 개발한 이유.
    브라우저의 호환성을 위해 개발
    오픈소스

- 특징
    모든 처리가 전부 함수
    getElementById() / innerHTML , vlaue // 자바스크립트에서는 함수와 속성이 나뉘어져있는데 제이쿼리는 다 함수로 바꿔버림
    jQuery()로 모든 로직을 처리함
    CSS ,DOM에 접근하기 쉽게 되어있다
    AJAX(비동기처리)를 편하게 사용하도록 도와준다
    ※ 특정함수를 사용 후에 반환타입이 다시 jQuery 객체이면 다시 JQuery 함수를 사용할 수 있다

- 사용방법
1) 라이브러리 임포트한다
   
2) 사용자가 명령을 개발한다
 <script scr="파일명">
    // 사용자 지정 명령
 </script>

 3) jquery 표현방법
    $('css 선택자') ==> 특정요소 선택시
    $('구조문서(태그)') ==> 특정 요소 선택시 ,jquery 객체 생성할 때
    $(콜백함수)  ==>  문서가 로드되면 실행할 함수를 기재
